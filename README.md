# Account

## Requirements

- Node.js >= 14

## How to setup dev environment?

- Copy `.env.dist` to `.env` and replace all envs ended with PLACEHOLDER suffix. Example:
  ```
    APP_URL=http://localhost:3000
    APP_API_URL=http://localhost:3001
    APP_API_PORT=3001
    DEV_DISABLE_TYPE_CHECK=1
    DEV_APP_PORT=3000
  ```
- Run `npm i`
- Run `npm run start`
